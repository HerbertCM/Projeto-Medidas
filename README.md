📖 Descrição:

Aplicação desenvolvida em Python com Streamlit para auxiliar no acompanhamento físico de alunos. O sistema permite cadastrar alunos, registrar medidas corporais, acompanhar histórico por data e enviar os resultados diretamente via WhatsApp.


🚀 Funcionalidades:

👤 Cadastro e gerenciamento de alunos

📏 Registro de medidas corporais vinculadas ao aluno

📅 Histórico de medições organizado por data

✏️ Edição de dados de alunos e medidas

📲 Envio automático das informações para o WhatsApp


🛠️ Tecnologias Utilizadas:

Python

Streamlit

Pandas


▶️ Como Executar o Projeto:

Clone o repositório:

git clone https://github.com/HerbertCM/Projeto-Medidas.git

Acesse a pasta do projeto:

cd Projeto-Medidas

Crie e ative um ambiente virtual (opcional):

python -m venv venv
venv\Scripts\activate

Instale as dependências:

pip install -r requirements.txt

Execute o projeto:

streamlit run app.py


🎯 Objetivo:

Facilitar o controle e acompanhamento da evolução física dos alunos, tornando o processo mais organizado, prático e profissional.
